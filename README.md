# Club de Yates Palmira Topolobampo

Sitio web oficial de la marina. Multipágina bilingüe (ES/EN), HTML estático.

## Estructura
- `index.html` + 7 páginas en español (raíz)
- `en/` — versión completa en inglés
- `img/` — imágenes optimizadas
- `sitemap.xml`, `robots.txt`, `llms.txt` — SEO e IA Search

Vercel (framework: Other, sin build). Dominio: marinapalmiratopo.com
